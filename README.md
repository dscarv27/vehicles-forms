Para iniciar:

```
git clone
```
------------------------------------------------------------
Backend (localhost:3001):
```
cd backend-fabrilan
```
```
yarn
```
```
docker compose up -d
```
```
yarn start
```
Prisma Studio (localhost:5555):
```
npx prisma studio
```
------------------------------------------------------------
Frontend (localhost:5173):
```
cd frontend-fabrilan
```
```
yarn
```
```
yarn run dev
```
